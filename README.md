# Professional Swarm V2

An autonomous software organism that audits, red-teams, and heals your codebase.

## Core Swarms
1. **Analysis Swarm:** Scans projects for improvement areas.
2. **Audit Swarm:** Deep-dives into code safety, performance, and structure.
3. **DevOps Swarm:** Prepares local sandbox environments.
4. **God Swarm:** Orchestrates testing and verification using embedded Quantum Engine & Mirofish-lite.

## Features
- GUI-based configuration (Streamlit).
- First-time setup wizard for API keys.
- WhatsApp notification integration.
- Automated Sub-agent and Auto-agent creation.
- Persistent Cron-based scheduling.
- Automated Changelog generation.

## Getting Started
- `main.py` - Entry point for the orchestrator.
- `gui/app.py` - Launch the configuration UI.
