# Aegis Swarm

The primary autonomous **Evolution Engine and Developer Workbench**.

Aegis Swarm is the high-level, production-grade evolution system designed to actively analyze, audit, red-team, and refactor your codebases. Unlike background sentinels, Aegis Swarm is a proactive workbench you use to evolve your software lifecycle.

## Hierarchy
- **Aegis Swarm (Primary):** The orchestrator for deep, active code evolution, quantum-inspired verification, and professional developer workflows.
- **Project-EVO (Sentinel):** The background immune system for passive security monitoring and basic PR-based self-healing.

## Core Swarms
1. **Analysis Swarm:** Scans projects for optimization & evolution hotspots.
2. **Audit Swarm:** Deep-dives into safety, security, and compliance.
3. **DevOps Swarm:** Provisions isolated sandbox environments.
4. **God Swarm:** Orchestrates evolution using the **Quantum Engine** and **Mirofish-lite** matrix entanglement logic.

## Features
- **Ascension Engine:** Multi-agent consensus-based deliberation.
- **Quantum-Mirofish Verification:** Advanced state-space collapse for evolutionary decisions.
- **GUI Dashboard:** Streamlit-based control center for API management and evolution tracking.
- **Token Optimization:** Integrated "Token Fracture" technology for cost-efficient intelligence.

## Getting Started
1. `pip install -r requirements.txt`
2. Configure environment variables.
3. Run `streamlit run gui/app.py` or execute `core/orchestrator.py` via CLI.
